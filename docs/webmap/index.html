About this map

Built from the SIRS public/private classifier pilot for Benin. Three views:

  • Rules only - deterministic name and tag rules.
  • Rules + model - rules plus the geography model filling in unknowns, conservative threshold.
  • Population-aware - same as above, with each department rebalanced to match the INFRE 2021-22 private rate.
