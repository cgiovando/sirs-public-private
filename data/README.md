# Data inventory

All inputs and derived outputs are gitignored. This file describes the data the pipeline expects to find at runtime, where it comes from, and how to point the scripts at it.

## Inputs (read-only, never modified by this pipeline)

### School locations + footprints

The primary input is a school-locations export combining Giga school registry coordinates with downstream campus/building/hazard layers from a country-by-country processing pipeline. The path to that export is set via the `SCHOOL_DATA_DIR` environment variable. The expected layout inside that directory is:

```
$SCHOOL_DATA_DIR/
├── 1. School locations/
│   └── WAFR_Schools_GIGA_ALL_dated_<DATE>.csv     # Giga export, school points with names + ADM1/ADM2
├── 2. Campus definition/                          # building footprints matched to schools
├── 3. Building characterization/                  # WSF evolution year, WSF3D height
├── 4. Hazards/                                    # flood depth raster, seismic raster
└── Taxonomy_GLOSI_Aerial_Images_<DATE>.xlsx       # GLOSI parameter targeting
```

The Giga school registry itself is open (https://giga.global). The packaged multi-country export with hazard layers is shared bilaterally; check with the data provider before redistributing.

If `SCHOOL_DATA_DIR` is unset, ingest scripts fail fast with a clear error.

### OpenStreetMap (`amenity=school`, `amenity=kindergarten`)

Pulled fresh per country via Overpass API by `stage-a-rules/ingest_osm.py`. Cached locally under `data/cache/overpass_<ISO2>.json` to avoid hammering Overpass. License: ODbL.

### INFRE annuaires (Benin)

National statistical annuaires from Benin's Institut National pour la Formation et la Recherche en Éducation, published as downloadable `.xls`. Used for department-level public/private school count priors (`stage_b_features/infre_priors.py`). Public, citation required.

For other pilot countries the equivalent source differs (Niger: MEP annuaire; Mali: CPS/MEN; etc.). Bilateral data sharing for per-school registers is still in progress for several countries; only aggregated counts are public today.

### GHSL (Global Human Settlement Layer) SMOD

Settlement model raster, 2020, 1km resolution. Source: JRC EC Open Data Portal. Used by `stage_b_features/ghsl_smod.py` for urban/rural categorization.

### Relative Wealth Index (RWI)

Meta Data for Good 2.4km grid. Public dataset. Used by `stage_b_features/rwi.py`.

## Derived (gitignored, regenerated by the pipeline)

- `data/cache/` - Overpass API responses, cached for reproducibility.
- `data/inputs/` - per-country joined Giga+OSM geojson, intermediate inputs that the pipeline produces but treats as "frozen for this run".
- `data/derived/` - Stage A labels, Stage C predictions, feature matrices, provenance JSONs.

All derived outputs are timestamped by run date in the filename and carry a sidecar `*.provenance.json` pinning the code commit, input file hashes, and rule-set version.

## Pointing the pipeline at your data

```sh
export SCHOOL_DATA_DIR=/path/to/your/wafr-schools-export
python -m stage_a_rules.ingest_giga --country BEN
python -m stage_a_rules.ingest_osm --country BEN
python -m stage_a_rules.join --country BEN
python -m stage_a_rules.label --country BEN --rules stage-a-rules/benin.yaml
```
